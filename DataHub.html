DataHub

Understand marketing and customers better with data

Data Visualization, Data Analysis

Client: Dailmer, Audi

ABOUT

Data Hub is a data analysis and data visualization platform that is designed for automotive manufacturers to mining data values by combining data collected from car sensors with a geographical information system. The result brings a better understanding of its customers and helpful for marketing strategy and driving experiment improvement.











Due to policy, I won’t be able to share more details about this project here. If you are interested in it, please contact me.




My Role



Product Design UX Design Research Communication Pitch Ideas for RFQ

My role in this project is to make use case proposals and pitch ideas for our clients. Deliver finalized proposals to the inside UI designer and engineering team. I made use cases based on figuring out potential values the data analysis can bring to the market, and how to visualize the potentials to a tangible data analyzing platform.

Overview

There are two main analyzing perspectives it provides to help the business: Firstly, it analyzes car owners’ portrait to help the marketing team to make precious strategiesSecondly,it helps the HMI service designers and engineers to understand better of users behavior by analyzing the usage of the HMI service.

Process

My challenge during this project is to communicate with clients and team members to find the insights of this project, then deliver design ideas within two weeks. Below shows the process of the project development.

  • STEP 1. Understand the available database field types by learning the data sample the client provided.
  • STEP 2. Research to find the potential use cases that can bring values to the clients. Then brainstorm with engineers to find out the data analysis methods, data visulization and algorithm that can be applied to this case.
  • STEP 3. Analyze the data requirement for each case with engineer team.
  • STEP 4. Draw wireframe of each cases.
  • STEP 5. Present use case proposal to client, got feedback, and did iteration.
  • STEP 6. Direct UI Design based on finalized use cases.
